CSS Selectors & Specificity

Selectors

2 different methods:

  • Ruleset
  • Inline

Declaration = Apply a style to a selected element

Ruleset Terms

Term Definition
Selector The beginning of the ruleset used to target the element that will be styled
Declaration Block The code in-between(and including) the curly braces ({}) that contains the CSS declarations
Property First part of declaration that signifies what visual characteristic of the element is to be modified
Value Second part of declaration that signifies the value of the property

Inline terms

Linking the CSS file

  • Link element within head of HTML file
  • Self-closing tag
  • Requires attributes: href = address, or path, to CSS file and rel = describes relationship between HTML file and CSS file because linking to CSS file, value set to 'stylesheet'.

Specificity

Best practice: Style elements while using lowest degree of specifiity so that if an element needs a new style, it is easy to override.

ids = most specific --> classes --> types

Adding more than one tag, class, or ID to a CSS selector increase specificity.
